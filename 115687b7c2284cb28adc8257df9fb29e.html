TeamViewer on Linux

UNIX

TeamViewer might not run expectedly on Arch Linux or it's child distributions. To successfully start TeamViewer, run the following command before launching the TeamViewer application:

sudo teamviewer --daemon enable

OR

sudo systemctl restart teamviewerd

Now launch TeamViewer :)
