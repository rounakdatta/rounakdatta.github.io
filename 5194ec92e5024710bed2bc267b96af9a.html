Pulling metrics from InfluxDB to Grafana

InfluxDB Grafana Scala Hotstar

We are currently using the InfluxDB query as:

SELECT "*" FROM "events-compacted"

This is giving the results from InfluxDB On Grafana. This can now be plotted on a Table.

⚠️

Watch-Out: https://github.com/influxdata/influxdb/issues/6459#issuecomment-216285152. There are some conventions to be followed while using the WHERE clause for InfluxDB queries.

For example, note the usage of single and double quotes:

SELECT * FROM "events-compacted" where cd='2019-06-20'

Comparing Data Size (Uncompacted vs Compacted)

SELECT state, cd, hr, dataSize FROM "events-compacted" where cd='2019-06-20' AND hr='17'
