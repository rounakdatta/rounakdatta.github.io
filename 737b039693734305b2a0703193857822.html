BFG-Repo-Cleaner

git Sensitive Data

BFG is an important tool developed to rewrite the git history and remove sensitive data committed in the past. The original documentation is at https://help.github.com/en/github/authenticating-to-github/removing-sensitive-data-from-a-repository.

  1. Clone a mirror copy (contains all the git history thereby) of the repository
    git clone --mirror https://github.com/rounakdatta/ainv-backend-go.git
  2. Download the BFG JAR in your system from https://repo1.maven.org/maven2/com/madgag/bfg/1.13.0/bfg-1.13.0.jar
    wget https://repo1.maven.org/maven2/com/madgag/bfg/1.13.0/bfg-1.13.0.jar
  3. We are demonstrating removal of passwords, credentials or any sensitive information. So, we need to construct the passwords.txt file as shown:
  4. We next allow the JAR to perform the actions.
    java -jar ~/tooling/bfg-1.13.0.jar --replace-text ~/passwords.txt ~/personal/ainv-backend-go.git
  5. Next, we navigate to the repository → reflog the changes → updated the remote.
    cd ~/personal/ainv-backend-go.git
git reflog expire --expire=now --all && git gc --prune=now --aggressive
git push
