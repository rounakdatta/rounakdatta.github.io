Setting file permissions using chmod

Permissions

CodeDescription
1can execute
2can write
4can read
3 (1 + 2)can (execute + write)
6 (2 + 4)can (write + read)
7 (1 + 2 + 4)can (execute + write + read)

Permission Syntax Format

PositionDescription
1what owner can
2what users in the file group (class) can
3what users not in the file group (class) can

Permissions Example

CommandDescription
chmod 600 fileowner can read, write
chmod 700 fileowner can read, write, execute
chmod 666 fileall can read, write
chmod 777 fileall can read, write, execute

List out all the files within a directory

find .
ls --recursive

List files starting with or ending with

find . -type f -name "abc*"
find . -type f -name "*.json"

List all the files in order of last modified

ls -t
# this can be well presented by combining it with -h as
# remember, h means human readable
ls -lht

Listing all the pacman-installed packages

(echo PACKAGE SIZE; \
  for A in /var/lib/pacman/local/*/desc; do
    (sed -n 2p $A; grep -E '^[0-9]{2,}$' $A) | tr '\n' ' '; echo
  done \
  | sort -nrk2) \
| column -t

Listing all the yaourt-installed packages

pacman -Qm

Prevent a pacman package from being updated

sudo nano /etc/pacman.conf

Add all the required package names under IgnorePkg = <packageName1> <packageName2>

Show the disk usage details

df -h
# h flag denotes human readable

Installing from tar.gz

tar -xzf softwarePackage.tar.gz

Remove all npm packages from a system (UNIX)

npm ls -gp --depth=0 | awk -F/ '/node_modules/ && !/\/npm$/ {print $NF}' | xargs npm -g rm

Getting the status of the last run command

echo $?

This gives output as 0 for true, and 127 (i.e. DEL) for false

Find the difference between the contents of two files

diff <firstFileName> <secondFileName>

List systemctl enabled services

systemctl list-unit-files | grep enabled

Show the time of execution for a command (Profiling)

time <command>

List the graphics card in the system

lspci -k | grep -A 2 -E "(VGA|3D)"

Not able to scroll in iTerm + SSH

CMD + R to refresh the screen in alternate mode

Command Line Debugging

  1. man command when executed without sudo is giving a strange error. With sudo, getting perfect output. Why?
    export MANPAGER=less
  2. sudo for yourt keeps getting timed out (asking for password again and again). How to solve this?
    sudo yaourt -Syyu

    These 3 flags need to be put in a file called ~/.yaourtrc:

    NOCONFIRM=1
BUILD_NOCONFIRM=1
EDITFILES=0
  3. WiFi requirement during CLI installation of Arch Linux

    Often it is recommended to connect to a wired LAN in the case of installation of a fresh copy of Arch Linux. However the steps to be followed to set up a network connection through WiFi during CLI installation of Arch Linux are as follows:

    1. sudo wifi-menu
    2. Select the wireless SSID and password.

      This will work good in the case of simple password-based network security. There might be the requirement of netctl or some advanced configurations in the case of using a more complex network.

    3. ping www.google.com
