ConfigFactory for Scala

Scala Configuration

Setting up

Using ConfigFactory in spark-shell

Assuming application.conf is available in the root directory, start the spark-shell as spark-shell --jars "config-1.2.1.jar" --files "application.conf"

import com.typesafe.config.ConfigFactory
import java.io.File
import org.apache.spark.SparkFiles

val config = ConfigFactory.parseFile(new File(SparkFiles.get("application.conf")))

Using ConfigFactory in Spark job

Usually application.conf is used as the configuration file, and it should be available in PATH.

import com.typesafe.config.ConfigFactory
val config = ConfigFactory.load()

Getting configuration parameters

config.getString("my.organization.project.name")
