Setting up MySQL Server

How To Install MySQL on Ubuntu 18.04 | DigitalOcean

MySQL is an open-source database management system, commonly installed as part of the popular LAMP (Linux, Apache, MySQL, PHP/Python/Perl) stack. It uses a relational database and SQL (Structured Query Language) to manage its data. The short version of the installation is simple: update your package index, install the mysql-server package, and then run the included security script.

Setting up phpmyadmin

How To Install and Secure phpMyAdmin on Ubuntu 18.04 | DigitalOcean

While many users need the functionality of a database management system like MySQL, they may not feel comfortable interacting with the system solely from the MySQL prompt. phpMyAdmin was created so that users can interact with MySQL through a web interface.

https://www.digitalocean.com/community/tutorials/how-to-install-and-secure-phpmyadmin-on-ubuntu-18-04

Following these steps carefully, you must be able to access phpmyadmin at http://<yourIP>/phpmyadmin

Debugging

phpmyadmin server is not loading

  1. sudo -H nano /etc/apache2/apache2.conf
  2. This line is to be included at any point in the file: Include /etc/phpmyadmin/apache.conf
  3. Restart the server using sudo service apache2 restart

Changing the MySQL (and phpmyadmin) password

  1. mysql -u root -p to open the mysql prompt.
  2. ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'password'; is the command to change the password. Replace password with your required text.

phpmyadmin Table View gives error count(): Parameter must be an array or an object that implements Countable

  1. Open up sudo nano /usr/share/phpmyadmin/libraries/sql.lib.php
  2. Goto line 614
  3. Replace 
((empty($analyzed_sql_results['select_expr']))
    || (count($analyzed_sql_results['select_expr'] == 1)
        && ($analyzed_sql_results['select_expr'][0] == '*')))

with 

((empty($analyzed_sql_results['select_expr']))
    || (count($analyzed_sql_results['select_expr']) == 1)
        && ($analyzed_sql_results['select_expr'][0] == '*'))

When you're unable to login due to password issues

sudo mysql -u root
SELECT User,Host FROM mysql.user;
# you'll see that root is there

# removing the root user first
DROP USER 'root'@'localhost';

# recreating the user with a custom password
CREATE USER 'root'@'%' IDENTIFIED BY 'customPassword';

exit
