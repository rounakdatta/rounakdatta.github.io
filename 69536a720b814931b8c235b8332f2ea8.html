libinput-gestures

UNIX

Why

Using macOS like gestures on Linux

Main repository: https://github.com/bulletmark/libinput-gestures

Enable libinput-gestures to be starting everytime on startup


libinput-gestures-setup autostart

Start libinput-gestures on demand

libinput-gestures-setup start

Stop libinput-gestures forcefully

libinput-gestures-setup stop

Restart libinput-gestures forcefully

libinput-gestures-setup restart

Configurations file location: .config/libinput-gestures.conf
