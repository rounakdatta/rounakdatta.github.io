How I met Scala's testing suite

Posted on Sun, Jul 28, 2019 Scala TDD Testing

Installation

scalatest needs to be added to the libraryDependencies in build.sbt as:

libraryDependencies += "org.scalatest" % "scalatest_2.13" % "3.0.8"
libraryDependencies += "org.scalatest" % "scalatest_2.13" % "3.0.8" % "test"

Getting started

Start by importing the FunSuite (functional suite) and other relevant libraries as:

import org.scalatest.FunSuite
import org.scalatest.BeforeAndAfter

Make sure you extend the new testing class with FunSuite and BeforeAndAfter

Heres' a simple test class:

import org.scalatest.FunSuite
import org.scalatest.BeforeAndAfter

class SimpleTests extends FunSuite with BeforeAndAfter {

  test("Creating an empty Parking Lot") {
    val emptyParkingLot: ParkingLot = new ParkingLot(0)
    assert(emptyParkingLot.findNearestEmptySlot() ==  -1)
  }
}

If some statements have to be compulsorily executed before the unit tests, use before:

before {
    testParkingLot = new ParkingLot(10)
  }
