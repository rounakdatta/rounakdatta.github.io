Case class

Case class in Scala is simply a class with 2 additional properties:

  1. Case classes hold immutable data and are therefore very useful for modelling such use cases
  2. Case classes are very helpful for pattern-matching - they solve the problem avoiding the need to iterate-and-match. An example is as
    case class Dog(breed:String,name:String,age:Int)

for(dog<-List(leo,fluffy,candy)){
	dog match{
		case Dog("Dogue de Bordeaux","Leo",2)=>println("Hi, Leo")
		case Dog("Lesser Indian Spitz","Fluffy",8)=>println("Hi, Fluffy")
		case Dog(breed,name,age)=>println("Hi, "+name)
	}
}

MrPowers/spark-fast-tests

A fast Apache Spark testing helper library with beautifully formatted error messages! Works with scalatest and uTest. The assertSmallDatasetEquality method can be used to compare two Datasets (or two DataFrames). The assertSmallDatasetEquality method can also be used to compare Datasets. The colors in the error message make it easy to identify the rows that aren't equal.
