UNIX

Basic operations on the apache2 server

Perform [start|stop|restart] operations on the Apache server. Any of the commands work:

sudo service apache2 <operation>
sudo /etc/init.d/apache2 <operation>

Note: systemctl [start|stop|restart|status] httpd.service should work fine for Arch Linux.

Change the default port for apache2 server

Use nano /etc/apache2/port.conf and change the default port from 80 to whatever.
