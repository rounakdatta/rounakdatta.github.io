Cleartext HTTP not permitted

This happens mostly when you're trying to connect to a server directly through the IP address (instead of a domain name). With Android 9, this is no longer permitted, and must be enabled within AndroidManifest.xml. You need to include android:usesCleartextTraffic="true" within the <application> tag.
